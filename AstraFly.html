AstraFLY
Sistema de gestion empresarial

ASTRAFLY es una solución de software empresarial desarrollada como proyecto final para la materia Desarrollo de Software III, enfocada en la gestión eficiente de vuelos y operaciones dentro de una empresa del sector aeronáutico. El sistema permite manejar de forma organizada los módulos de vuelos, reservas, clientes, destinos, tripulación y aeronaves, ofreciendo al usuario una experiencia clara y fluida para llevar el control de los procesos clave en tiempo real. Su diseño modular permite registrar, editar y consultar información con facilidad, favoreciendo la toma de decisiones y la automatización de tareas administrativas.

El proyecto fue desarrollado utilizando C# bajo .NET Framework, empleando una arquitectura en capas que garantiza la separación de responsabilidades y facilita el mantenimiento del código. Se utilizaron tanto aplicaciones Windows Forms como una interfaz web desarrollada con .NET, HTML y CSS, asegurando así una cobertura multiplataforma. Además, se integró una WebAPI propia, que permite la comunicación entre las diferentes capas del sistema y la sincronización de datos en tiempo real entre las aplicaciones de escritorio y web.

Para el almacenamiento y seguridad de los datos se utilizó SQL Server, incorporando técnicas de cifrado a nivel de base de datos para proteger la información sensible, como datos personales de clientes y usuarios del sistema. También se implementó el manejo de archivos JSON para la configuración dinámica del sistema y la exportación de datos, lo que facilitó la interoperabilidad y la flexibilidad del entorno. La trazabilidad del sistema y el registro de eventos se gestionaron mediante Log4net, lo cual permitió una supervisión técnica eficaz durante las pruebas y simulaciones.
