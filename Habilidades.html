Experiencias y Habilidades

Lenguajes de Programación

C#

Nivel medio. He desarrollado aplicaciones de escritorio con Windows Forms y .NET Framework, implementando interfaces gráficas, manejo de eventos y lógica orientada a objetos. Familiarizado con la integración de librerías como Log4Net para registro de eventos y con la creación de servicios WebAPI.

Python

Nivel medio. He trabajado en scripts para automatización, análisis de datos y desarrollo de interfaces gráficas usando Tkinter y CustomTkinter. Implementé proyectos de extracción de texto con PIL, generación de laberintos con Pygame y herramientas de seguridad basadas en Scapy y Paramiko.

C++

Nivel medio. Experiencia en programación de algoritmos y estructuras de datos (listas enlazadas, pilas). He implementado funciones para ordenar arreglos y calcular estadísticos (media, moda, mediana) utilizando punteros, así como proyectos de simulación sencilla en consola.

HTML & CSS

Dominio de construcción de páginas estáticas, diseño responsivo y uso de frameworks ligeros. He creado prototipos educativos y portafolios web minimalistas sin JavaScript, aprovechando únicamente HTML5 y CSS3 para lograr interfaces limpias y adaptables.

SQL

Competencia en MySQL y SQL Server. Diseño de bases de datos relacionales, normalización de tablas, creación de consultas SELECT complejas, uso de funciones de ventana (window functions), procedimientos almacenados y optimización de índices.

VHDL

Conocimiento en diseño de circuitos digitales y simulación con herramientas como EDA Playground y Tinkercad. He implementado componentes básicos (sumadores, registros) y pruebas unitarias para proyectos académicos en electrónica digital.

Tecnologías y Herramientas

  • SQL Server: Administración de bases de datos, diseño de esquemas, implementación de cifrado a nivel de base de datos y creación de respaldos.
  • MySQL: Creación y consulta de bases de datos, procedimientos almacenados, funciones y triggers en entornos LAMP y WAMP.
  • Log4Net: Configuración de registros de eventos y errores en aplicaciones .NET para monitoreo y diagnóstico en tiempo de ejecución.
  • WebAPI (.NET Framework): Desarrollo de servicios RESTful que exponen endpoints para interacción con aplicaciones de escritorio y web, manejo de JSON y autenticación sencilla.
  • .NET Framework: Plataforma para desarrollo de aplicaciones de escritorio con Windows Forms, manejo de excepciones, diseño orientado a objetos y uso de bibliotecas estándar.
  • Windows Forms: Creación de interfaces gráficas con componentes como TextBox, Button, DataGridView y manejo de eventos para proyectos de escritorio en C#.
  • Tkinter y CustomTkinter: Construcción de interfaces gráficas en Python con temas oscuros y claros, uso de widgets avanzados (Notebook, ScrolledText, MessageBox) y diseño modular de ventanas.
  • Linux: Experiencia en entornos de desarrollo basados en Ubuntu y Debian; administración básica de usuarios, instalación de paquetes y configuración de scripts.
  • Python Imaging Library (PIL): Manejo y manipulación de imágenes dentro de aplicaciones, redimensionado, conversión de formatos y almacenamiento en base de datos.
