Gestor de Recetas

Esta aplicación de escritorio está desarrollada en Python, utilizando CustomTkinter y la biblioteca PIL para ofrecer una interfaz gráfica intuitiva y moderna. Al iniciar, el usuario accede mediante un sistema de login seguro. Una vez validado, aparece un menú principal con opciones claras para insertar, ver, editar o eliminar recetas. Cada módulo maneja de forma organizada toda la lógica de negocio: inserción de datos, almacenamiento de imágenes y procesamiento de ingredientes y pasos.

Funciones Clave

Inserción de Recetas: Permite agregar una nueva receta ingresando nombre, descripción, tiempo, complejidad y fecha. El sistema facilita cargar y copiar automáticamente la imagen asociada para guardarla localmente. A través de un formulario dinámico, el usuario introduce ingredientes (nombre, cantidad y unidad) y define cada paso de preparación en orden, construyendo internamente una lista encadenada de pasos.

Visualización y Navegación: Ofrece un filtro por nivel de complejidad para listar recetas existentes. Al seleccionar una receta, se despliegan sus detalles (imagen, descripción, comentarios, duración, complejidad e ingredientes) y, mediante botones de “anterior” y “siguiente”, se navega paso a paso a lo largo de la preparación, facilitando la consulta ordenada de cada etapa.

Edición y Eliminación: El módulo de edición permite filtrar recetas por complejidad, elegir una receta y modificar todos sus campos: nombre, descripción, tiempo, complejidad, comentario, imagen, ingredientes y pasos. Cada cambio se guarda de forma íntegra, actualizando la fuente de datos subyacente. De manera similar, la opción de eliminación confirma la receta a borrar y la remueve del sistema, actualizando inmediatamente la lista de recetas disponibles.

Tecnologías y Arquitectura

Interfaz: La capa de interfaz se construye con CustomTkinter, aprovechando temas oscuros y claros, botones con iconos y cuadros de texto estilizados.

Gestión de Imágenes: Para la gestión de imágenes, se usa PIL (Python Imaging Library), lo que posibilita redimensionar y renderizar fotos de recetas dentro de la aplicación.

Lógica de Datos: La lógica de datos (insertar, obtener, actualizar y eliminar recetas, ingredientes y pasos) está encapsulada en clases como RecetaFunciones y ListaPasos, que se comunican con el almacenamiento subyacente (por ejemplo, XML o base de datos).

Diseño Modular: El diseño modular separa claramente la ventana de login, el menú principal y cada pantalla funcional (insertar, ver, editar, eliminar), facilitando futuros mantenimientos o ampliaciones.

En conjunto, este Gestor de Recetas ofrece un flujo sencillo y completo: el usuario inicia sesión, selecciona la acción deseada, trabaja con formularios detallados para ingresar o modificar información, y navega de manera clara por toda la receta, desde sus ingredientes hasta cada paso de preparación. Gracias a la combinación de CustomTkinter y PIL, la experiencia resulta ágil, visualmente atractiva y adecuada tanto para usuarios finales como para demostraciones en portafolio.
