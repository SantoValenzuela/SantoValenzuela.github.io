Este es el ultimo proyecto
en el que he trabajado

ASTRAFLY es una aplicación integral desarrollada como proyecto final para la asignatura Desarrollo de Software III, con el propósito de optimizar la gestión operativa de una empresa del sector aeronáutico. El sistema incluye módulos completos para el manejo de vuelos, reservas, clientes, destinos, tripulaciones y aeronaves, permitiendo una administración ágil y centralizada de todos los procesos clave. Se diseñaron e integraron herramientas para el registro y consulta de información, asignación de horarios y control de operaciones, logrando una solución eficiente y profesional.

El desarrollo se realizó en C# utilizando .NET Framework, con una arquitectura robusta que incluye una WebAPI para la comunicación entre aplicaciones, una interfaz web desarrollada con .NET y el uso de SQL Server como sistema gestor de base de datos. Se implementó también Log4net para la gestión de logs y trazabilidad. Este proyecto pone en práctica conocimientos avanzados en diseño modular, consumo de servicios web, programación orientada a objetos y gestión de bases de datos, demostrando la capacidad para construir soluciones reales de software empresarial desde cero.

El proyecto ASTRAFLY no solo representa un logro académico, sino también una experiencia práctica significativa en el desarrollo de software, reflejando la capacidad para enfrentar desafíos reales en el ámbito tecnológico y empresarial.
